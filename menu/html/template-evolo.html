Template Evolo
Keterangan

Dibangun dengan Bootstrap 4, HTML / CSS, dan JS, fitur slider gambar untuk logo mitra, slider teks untuk testimonial, detail lightbox untuk informasi tambahan, video lightbox, halaman tambahan seperti ketentuan syarat dan kebijakan privasi. Template memiliki gaya desain ilustrasi yang indah dan termasuk formulir kontak yang berfungsi penuh.

Fitur
  • Slider gambar dengan pratinjau lightbox besar
  • Munculan lightbox untuk detail video, gambar dan teks
  • Formulir kontak yang berfungsi, ramah GDPR
  • Menu navigasi dropdown
  • Pilih antara file gambar dan logo teks
  • Halaman tambahan: ketentuan syarat dan kebijakan privasi
  • Google Fonts dan integrasi Font Awesome
  • Desain responsif yang bekerja sangat baik pada perangkat apa pun
  • Gambar atau logo teks
  • Kode valid W3C diuji di semua browser utama
Dibangun Dengan
  • Bootstrap 4, HTML / CSS, JS, PHP
  • Google Fonts, Font Awesome
Persyaratan
  • Pengodean HTML / CSS dan keterampilan mengedit gambar
