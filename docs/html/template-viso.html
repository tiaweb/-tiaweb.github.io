Keterangan

Dibangun dengan Bootstrap 4, HTML / CSS, dan JS, fitur formulir pengajuan CV yang berfungsi dengan lampiran file pdf / kata, kisi portofolio untuk daftar pekerjaan, morphtext, slider kartu untuk testimonial, detail lightbox, angka statistik untuk nilai-nilai penting, navigasi dropdown, halaman tambahan untuk rincian pekerjaan, layanan, harga, tentang, artikel, ketentuan, privasi, dan kontak.
Templat ini memiliki desain cahaya yang bersih dengan ilustrasi modern yang mengatur nada yang tepat bagi pengunjung untuk menelusuri tawaran pekerjaan dan mengirimkan aplikasi CV mereka melalui formulir.
